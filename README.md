# MovieReviewSite


# MovieReviewSite/WebContent(웹 구조 기본 틀)


## 📂 설명
영화 리뷰 웹사이트용 기본 웹 구조/ JSP 기반 동작 확인용 환경을 포함


## ✅ 실행 방법 (Eclipse + Tomcat)
1. Eclipse에서 Dynamic Web Project로 Import
2. 본인 Tomcat 등록 후 Servers 뷰에서 프로젝트 추가
3. 브라우저에서 http://localhost:8080/MovieReviewSite/index.jsp 접속
   (포트는 각자 설정에 따라 조정)
   

## 📁 포함된 JSP
- index.jsp: 메인 페이지
- login.jsp: 로그인 폼
- register.jsp: 회원가입 폼
- movieList.jsp: 전체 영화 목록
- detail.jsp: 영화 상세 정보
- mypage.jsp: 사용자 마이페이지
- admin.jsp: 관리자 기능
- web.xml: welcome 파일 및 기본 서블릿 설정
